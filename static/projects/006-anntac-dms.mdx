Worked on the user interface part of AnnTac Document Management System using
[Microsoft Silverlight](https://www.microsoft.com/silverlight/) that runs on
both Windows and web browser. The application aims to help organisational
digitalisation by turning paper-based documents into digital one. The system
offered a physical scanner, auto document organisation, and document management
user interface. The user inferface offers document preview which supports
various kind of extension, document annotation, full-text search, and tree view.
